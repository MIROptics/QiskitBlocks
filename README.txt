QiskitBlocks [qiskitblocks_game]
================================
Teaching quantum computing using Qiskit in a block world
========================================================

To use this game with the Minetest engine, insert this repository as
	/games/qiskitblocks_game

The Minetest engine can be found in:
	https://github.com/minetest/minetest/

Compatibility
--------------
The QiskitBlocks github master HEAD is generally compatible with the github
master HEAD of the Minetest engine.
